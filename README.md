# Arivora Global 🌍

**Your Comprehensive Educational Hub for Science, Technology, History, Research & News**

## 🎯 Overview

Arivora Global is a modern, professional educational platform built with Next.js and Tailwind CSS. It provides comprehensive content across multiple domains:

- 🔬 **Science Hub** - Scientific discoveries and research
- 💻 **Technology & Innovation** - Latest tech trends
- 🧬 **Physiology** - Human body and biological systems
- 📚 **History & Research** - Historical events and research
- 📰 **News & Updates** - Global news and updates

## ✨ Features

### 🎨 Modern Design
- Professional UI with dark/light mode support
- Responsive design for all devices
- Smooth animations and transitions
- Gradient accents with cyan and blue theme

### 🔧 Technical Stack
- **Next.js 14** - React framework
- **Tailwind CSS** - Utility-first CSS
- **TypeScript** - Type safety
- **Lucide React** - Icon library
- **Next Themes** - Dark mode support

### 🚀 Performance
- Static site generation
- Fast page loads
- SEO optimized
- GitHub Pages deployment ready
